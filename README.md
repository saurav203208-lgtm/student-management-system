# Student Management System

A REST API built using Node.js, Express.js, and MongoDB.

## Features
- Create Student
- Get All Students
- Get Student By ID
- Update Student
- Delete Student

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## API Endpoints

POST /api/students

GET /api/students

GET /api/students/:id

PUT /api/students/:id

DELETE /api/students/:id

## Run Project

```bash
npm install
npm start
